Easy and simple way to embed ALL IN BOX contest on WordPress.

== Description ==

All In Box plugin helps you add your contests to your WordPress website in an easy and simple way.

In order to use this plugin you must have an account on https://apps.allinbox.fr and create a contest.

The data of contest participants will be automatically transferred to your ALL IN BOX account.

You can find the terms of use at this address:
https://app.allinbox.fr/cgu/

and the privacy policy at this address:
https://app.allinbox.fr/rgpd/

== Features ==

1. Import ALL IN BOX contest to WordPress
2. Customize embed widgets via shortcode.
3. Easy to use widget customizer for customizing and generating embed widget shortcodes.

== Installation ==

1. Upload and activate the plugin (or install it through the WP admin console)
2. That's it, it is ready to use!


== Shortcode ==

Embed All In Box page on WordPress with:

`[allinbox code="MYKEY" height="1000"]`

== Frequently Asked Questions ==

= How do I display my contest on pages? =

Add `[allinbox code="MYKEY" height="1000"]` shortcode to any page you want to display the contest on.
